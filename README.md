# Salesforce Domain Applications (Banking & Education)

## Overview
This repository contains Salesforce-based applications developed for **Banking** and **Education** domains.  
The primary focus of these projects is **backend system design**, including data modeling, access control, and workflow automation.

Due to Salesforce org access restrictions, a live hosted demo cannot be shared. Instead, detailed screenshots and documentation are provided to demonstrate the implementation.

## Backend Concepts Demonstrated

- **Data Modeling**
  - Custom objects and relationships
  - Master-Detail and Lookup relationships
  - Junction objects for many-to-many relationships
  - Schema Builder diagrams (ER-style)

- **Security & Access Control**
  - Custom profiles (Cashier, Manager, Help Desk, Trainer, etc.)
  - Object-level and field-level security
  - Role-based data access

- **Automation & Business Logic**
  - Record-triggered and scheduled flows
  - Conditional logic using decisions
  - Loop-based record processing
  - Automated emails and validations
